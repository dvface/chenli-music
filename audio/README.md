## 如何获取 MP3 文件

你可以通过以下合法途径获取这些歌曲的 MP3 文件：

1. **网易云音乐 / QQ 音乐下载**（开通会员后可下载，部分歌曲需解密转码）
2. **购买数字专辑** — Apple Music / 网易云音乐数字专辑
3. **CD 抓轨** — 如果你拥有实体 CD，可用 iTunes/Windows Media Player 转为 MP3
4. **iTunes Store 购买** — 购买后可直接获得无 DRM 的 AAC 文件

⚠️ 请勿使用盗版渠道获取音乐，尊重音乐人版权。

## 文件要求

- 格式：MP3（推荐）或 AAC
- 比特率：128kbps 以上，推荐 320kbps
- 大小：建议每首 3–10 MB
